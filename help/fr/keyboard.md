Modern Commander Desk v0.3.3 - Aide clavier

Tab: changer le panneau actif
Enter: ouvrir le fichier/dossier
Backspace: dossier parent
Ctrl+R: actualiser le panneau actif
F1: afficher cette aide
F2: renommer
F3: prévisualiser le fichier sélectionné
F4: éditer un fichier texte/code
F5: copier vers l'autre panneau
F6: déplacer vers l'autre panneau
F7: créer un dossier
F8/Delete: corbeille douce
F10: quitter
