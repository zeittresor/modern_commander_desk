# helpers

Optional helper tools can be placed here or installed globally and referenced from `preview_handlers/*.json`.

Useful candidates:

- `mpv` for audio/video/tracker modules
- `vlc` for audio/video
- `ffplay` from FFmpeg for audio/video
- `openmpt123` for Amiga MOD/XM/S3M/IT playback
- `timidity` or `fluidsynth` for MIDI playback

The core app does not bundle these tools to keep the file manager small and legally simple across Windows and Linux.
